# 🎂 Pipeline 40 — Birthday Dashboard

A playful, interactive **Azure Data Factory–inspired dashboard** built with React, created for a special 40th birthday 🎉

👉 Live demo: https://justynaaleks.github.io/pipeline40/

---

## 🚀 Features

✨ **Azure-like UI**

* Topbar, sidebar, tiles and layout inspired by Microsoft Azure Data Factory

🎯 **Interactive pipeline**

* Clickable stages (Ingest → Transform → Validate → Deploy)
* Visual progress with animated states

🖼️ **Photo gallery**

* Dynamic image loading
* Click to enlarge + navigate with arrows
* Smooth layout and responsive grid

🍫 **Chocolate terminal (easter egg)**

* Simulated terminal with typing effect
* Dependency check with humor
* Animated cursor + staged logs

🎈 **Fun details**

* Floating balloons
* Search bar easter eggs
* Micro-interactions on tiles

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🎨 CSS (custom styling, no frameworks)
* 🔁 React Router
* 🎯 Fluent UI Icons

---

## 📦 Deployment

This project is deployed using **GitHub Pages**.

Key setup:

* Vite `base: '/pipeline40/'`
* Production build from `/dist`
* Static hosting via GitHub Pages
